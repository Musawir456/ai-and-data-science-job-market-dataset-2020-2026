# 🤖 AI & Data Science Job Market Dataset (2020–2026)

A comprehensive dataset and exploratory analysis of the AI and Data Science job market, covering **10,345 job postings** from **2020 to 2026** across multiple countries, industries, and experience levels.

---

## 📁 Repository Contents

| File | Description |
|------|-------------|
| `AI Job Market Dataset.csv` | Main dataset with job listings and attributes |
| `ai-and-data-science-job-market-dataset-20202026-et.ipynb` | Jupyter Notebook — ETL, EDA & Visualization |

---

## 📊 Dataset Overview

The dataset captures real-world trends in AI and Data Science hiring across multiple dimensions:

- **Rows:** 10,345 job records
- **Columns:** 19 features
- **Time Period:** 2020 – 2026
- **Countries:** USA, UK, Canada, Australia, Germany, India, Singapore, and more

### 🗂️ Features

| Column | Description |
|--------|-------------|
| `job_id` | Unique identifier for each job posting |
| `job_title` | Role title (e.g., Data Scientist, AI Engineer, ML Engineer) |
| `company_size` | Size of the company (Startup, Medium, MNC, Enterprise) |
| `company_industry` | Industry sector (Technology, Finance, Healthcare, Retail, etc.) |
| `country` | Country where the job is located |
| `remote_type` | Work mode — Remote, Hybrid, or Onsite |
| `experience_level` | Entry, Mid, or Senior level |
| `years_experience` | Number of years of experience required |
| `education_level` | Required education (Bachelor, Master, PhD) |
| `skills_python` | Python skill required (0/1) |
| `skills_sql` | SQL skill required (0/1) |
| `skills_ml` | Machine Learning skill required (0/1) |
| `skills_deep_learning` | Deep Learning skill required (0/1) |
| `skills_cloud` | Cloud skill required (0/1) |
| `salary` | Annual salary in USD |
| `job_posting_month` | Month the job was posted |
| `job_posting_year` | Year the job was posted |
| `hiring_urgency` | Urgency level — Low, Medium, or High |
| `job_openings` | Number of open positions for the role |

## 📓 Notebook

The Jupyter Notebook performs a full **ETL → EDA → Visualization** pipeline:

- **ETL:** Data loading, type casting, missing value checks, and cleaning
- **EDA:** Statistical summaries, distributions, salary analysis by role/country/experience
- **Visualization:** Charts and graphs for trends, skill demands, remote work patterns, and hiring urgency

## 🚀 Getting Started

```bash
git clone https://github.com/Musawir456/ai-and-data-science-job-market-dataset-2020-2026.git
cd ai-and-data-science-job-market-dataset-2020-2026
```

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

Run the notebook:

```bash

```
